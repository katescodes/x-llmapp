## 输出要求

### 1. 格式要求
- **输出HTML格式的章节内容**
- **不要输出章节标题，只输出正文内容**
- **必须使用标准HTML标签**，不要使用Markdown语法

### 2. HTML标签规范
- 使用 `<p>` 标签包裹段落
- 使用 `<h3>`、`<h4>`、`<h5>` 标签显示小标题
- 使用 `<ul>`、`<li>` 标签显示列表
- 使用 `<strong>` 标签加粗重点内容
- 使用 `<table>`、`<tr>`、`<th>`、`<td>` 标签展示表格数据
- 使用 `<br>` 标签换行（如需要）

### 3. 内容要求
- 内容必须完整、专业、符合评审标准
- 结构清晰，逻辑严密，语言规范
- 紧扣申报要求，突出项目优势
- 至少3-6段内容，充分论述

### 4. 图片引用规则
{{#if has_images}}
- 如果用户资料中包含相关图片，请在合适位置插入图片占位符
- **格式**：`{image:图片文件名}`
- **位置**：应在独立的段落标签中，例如：`<p>{image:企业厂房外观.jpg}</p>`
- **说明**：图片后可添加说明文字
{{/if}}
