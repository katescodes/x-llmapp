## 输出要求

### 1. 格式要求
- 输出HTML格式的章节内容

- 不要输出章节标题，只输出正文内容
- **如果用户明确要求表格格式，必须使用HTML表格标签**（`<table>`、`<thead>`、`<tbody>`、`<tr>`、`<th>`、`<td>`）

### 2. 内容要求
- 内容至少{{min_words}}字，分为3-6段
- 确保内容完整、逻辑清晰、论述充分

### 3. 章节类型指导
根据标题类型生成合适内容：

- **格式类章节**（投标函、授权书等）
  → 生成对应的格式范本，包含必要的格式要素

- **技术方案类章节**
  → 详细描述技术路线、实施方法、保障措施
  → 体现技术可行性和先进性

- **商务类章节**
  → 说明报价依据、优惠措施、付款方式
  → 展现商务优势和合理性

- **公司/业绩类章节**
  → 充分利用企业资料展示实力
  → 突出相关项目经验和成功案例
